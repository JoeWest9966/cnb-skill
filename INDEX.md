# CNB 技能索引

根据用户的需求，匹配并调用最合适的 CNB 子技能。

## 技能列表

### API 与知识库

- **cnb-api** — 需要调用 CNB 平台 API、查询或操作平台资源时
- **cnb-knowledge-base** — 用户询问 CNB 平台使用方法、功能介绍、操作指南时

### 代码与提交

- **code-commit** — 需要修改代码、实现功能、修复 Bug 并提交推送时
- **code-review** — 用户请求 review、评审、审查代码或 PR 时

### PR 相关

- **pr-diff** — 需要查看 PR 代码变更、获取文件差异时
- **pr-summary** — 用户请求总结 PR 变更、概述改动内容时

### CI/CD

- **cnb-pipeline** — 需要生成或修改 .cnb.yml 流水线配置、流水线失败诊断、CI 性能优化时

### 工具

- **upload-attachment** — 需要在 Issue 或 PR 中上传文件或图片时
- **text-path-converter** — 读取 Issue 或 PR 内容时需要解析相对路径
- **npc-search** — 用户想查找、搜索、推荐 NPC 角色，或不知道用哪个 NPC 时
- **npc-delegate** — 用户在 Issue 或 PR 中 @npc 让其完成开发任务时
- **tapd-resource-fetcher** — 用户提供 TAPD 链接或提及 TAPD 资源时
- **markitdown** — 将 PDF、Office 文档、图片、网页等多种格式转换为 Markdown 时
