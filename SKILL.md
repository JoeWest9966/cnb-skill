# CNB Skill

CNB 平台操作技能集合。

## 子技能

| 技能 | 说明 |
|------|------|
| cnb-api | CNB 平台 API 交互 — 仓库、Issue、PR、流水线操作 |
| cnb-knowledge-base | CNB 平台文档查询 — 功能介绍、操作指南 |
| cnb-pipeline | 流水线配置与诊断 — .cnb.yml 编写、构建失败排查 |
| code-commit | 代码编写、提交推送、创建 PR |
| code-review | 代码评审 |
| pr-diff | PR 代码变更查看 |
| pr-summary | PR 变更总结 |
| npc-delegate | NPC 委派任务 |
| npc-search | NPC 角色搜索 |
| tapd-resource-fetcher | TAPD 资源获取 |
| text-path-converter | Issue/PR 内容中路径解析 |
| upload-attachment | Issue/PR 上传文件或图片 |

## 使用说明

每个子技能目录包含独立的 SKILL.md，可直接阅读。需要深入了解时查看各目录下的 references/ 子目录。
